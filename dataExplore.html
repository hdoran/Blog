Basic Data Exploration and Variance Estimation

Published January 28, 20225 min read

Basic Data Analysis

Sometimes we just need a tool to open and explore our data. PACER provides a way to do this that is fast and with many options for data exploration. In this blog post I will walk through a simple and effective way to read any data file into the web application and do some basic data exploration. This module in PACER is perhaps the easiest to use because data can be in any format and the options are simple point and click. Even though you can use any data file, if you want to replicate the example I use for this tutorial you can grab it here. This file happens to be an R data file, but it doesn't matter because you can import any type of data into PACER, just make sure to use the radio buttons for ".csv/other" for different formats.

Data Exploration

The first thing we will do is head over to "Data Tools" in the top navigation bar and read in our data as follows. On the "Data" tab use the browse button to locate and read in your data. The upload bar will read in the file and I immediately clicked on the "Data View" tab in the main panel which shows our data.

The first thing to point out is this is an INTERACTIVE data table. I'll show you a few quick features. Note, we read in an R data file, but using the buttons at the top we can export the data to any format right away! First, if the data are too big and you want to hide some columns on the screen, click the "Column Visibility" button and a list of all the columns appear. Click the ones you no longer want and they will be hidden. Don't worry, you can always bring them back if you just click them again after they are hidden.

Here is another very cool feature. At the bottom of the table are boxes to filter your data. Just start typing and the filtering will happen in real time. This is not case sensitive. For the numbers, you will get a "slider" to you can filter a range. But, if you want a specific value in a box with numbers just type into the box 200...200 as an example and this will give you all rows with that exact value or you can type in the range you specifically want. Also, if you click and drag the column names, you can move the columns around to be in different positions.

Let's now move over to the "General" tab in the sidebar and use the drop down menu to choose "Get Descriptive Statistics". Choose a variable (must be a numeric variable) and click the button to run. Now, on the "Result View" panel you will see the summary of the variable. Very easy!

Let's do one "complex" thing. Use the drop down menu to choose "Get Standard errors" and you will see a list of options for estimating standard errors from complex sample designs. Here you just choose the variable you want to get the variance of and let's choose "cluster" and choose our cluster variable. Now simply click run and the table appears with the standard error, design effect, intraclass correlation and so on.

Last thing for now is to simply show the easy way to get visual displays. Head over to the "Graphics" tab and choose a plot and choose your variables like I have in this screen shot. Click to run the graphic and there you go.

Some Notes

I have shown only a few very small features. You use the drop down menus to choose many other ways to explore your data. Some of the learning comes through exploration so try using some of the other features and let me know if you have any thoughts or questions.
