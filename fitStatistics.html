IRT Fit Plots

Visual displays of data are the most powerful tools we can use to assess the degree to which our data fit the model. PACER makes it easy to read in some data after calibration and produce visual displays of the data to evaluate model fit. This tutorial provides sample data files to use for practicing and a high-level overview to get you started. Running these analyses requires three data files. The first file simply holds the item response data in a wide format as shown in this sample. The second file holds the model parameters as shown here and the third file holds the ability estimates as shown here .

As a reminder there is help in the app also. Look at the navigation bar at the top right and click help. Then, use the drop down menu to choose "Fit Plots" as shown here

Read in Data

The first thing we will do is head over to "Fit Statistics" in the top navigation bar and read in our data as follows. First, on the "Data" tab use the browse button to locate and read the first file called "theta.csv" and use the drop down menu to choose "theta" as the variable we need to use.

Now let's read in the item data and the item parameters as shown in this screenshot.

That's it! Now, you'll see a drop down menu appear in the main panel but no item plots will be produced until you choose one from the menu as shown below. The first two plots are based on the residuals and show the degree to which the empirical data fit the model-based curves from calibration. Binary items show the single ICC and polytomous items (based on the GPCM) show one curve for each score point. By default the standard error bars are based on the 95% confidence intervals using the Clopper Pearson intervals.

The second two plots are somewhat unique. The first plot is a "violin" plot showing the distribution of abilities at each score point. It's sort of a visual representation of the point biserial and here we hope that ability increases generally over score points. The second plot is an adaptation of theoretical QQ plots applied to IRT. It is based on the work described here.

You can now move over to the "Options" tab and make any adjustments to the pots you like. For example, if you prefer a 68% confidence interval, change the ".95" to a ".68" and things will be automatically adjusted on the screen. The polytomous items might seem a little crowded on the screen so use the "Number of Panels" option to increase the number and space things out as you prefer.

Last, if you head over to the "Q3 Options" tab, you can select the items you want to include in the Q3 analysis. This is useful to evaluate local independence of the items. Simply select some, or all, of your items and click the "Calculate Q3 Statistic" button. It may take a moment, but it will run and report back if any of the items are correlated at a value that exceeds a threshold. The default threshold is .2, but this can be set to any value you like.

Some Notes

You can see there is a navigation bar for "Q1" etc. Click over to those to get Yen's Q1 statistics and other visual displays on the screen.
