Published February 5, 202210 min read

Item Analysis

Classical item analysis is generally a good first approach for the analysis of item response data. It gives a useful collection of descriptive statistics that indicate the general behavior of the items from the samples collected. PACER has a large collection of psychometric tools for classical item analysis and to create dashboard views. This tutorial will walk through an example of how to run item analyses with the sample here. Download this file to your computer and you'll be able to replicate the analyses done in this tutorial.

Screen Capture Tutorial

A picture is worth a thousand words, but video is sometimes better. Here's a snippet screen capture showing how to interact with PACER and run the item analysis module.

Read in and Preview Data

The first thing we will do is head over to "Data Tools" in the top navigation bar and read in our data as follows. First, on the "Data" tab use the browse button to locate and read the first file called "itemDat.csv" if you're using the sample data or read in any data file you want.

Once you read in the data, you can preview the data by clicking on the "Data View" tab in the main panel. Here you will see this data table that you can interact with in real time. You can hide columns using the column visibility button, you can filter in real time using the boxes at the bottom of the table, you can grab the column header with your mouse and move columns around, you can export your data with the buttons, etc. Again, PACER is intended to be intuitive and work like any other web application so no real software learning is expected.

Estimate Reliability

Now head over to the "Psychometrics" tab in the sidepanel and estimate the reliability as shown in this figure. We simply use the menus to highlight which items we want to include, and click "Compute Reliability" when you are ready. Ths will output the following table. The leave one out table is showing what the coefficient alpha wold be if that particular item were excluded.

Run Classical Item Analysis

Now, on the side panel, use the drop down menu as shown below to choose "Classical Item Analysis" and again highlight the items you want included in the analysis. Click "Compute Classical Item Statistics" and on the "Results View" panel you will see the following output. The columns are the mean item response, standard error of the mean, the point-biserial (which can be corrected or not), the biserial (or polyserial for polytomous items), the maximum observed score in the data, the relative mean (item mean divided by max score), the item standard deviation, the omit count, the number of counted responses, and the size within each possible score point for each item.

View Item Analysis Dashboard

You can now move over to the "IA Dashboard" tab and you will see a summary dashboard along with a plot you can change. Choose the variable you want to view and the dotplot will create an ordered view of that statistic for each item. This is an interactive plot, so you can zoom in/out, hover over dots to see data, export the plot and more.

Some Notes

This module runs descriptive statistics, but other IRT-based fits are also available. See the blog post on IRT Fit Statistics for tutorials on additional ways to preview item response data.
