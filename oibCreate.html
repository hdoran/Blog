How To Estimate Response Probabilities Using PACER

Published January 17, 20225 min read

Response Probabilities

Response probabilities, or RP values, are used in psychometrics in multiple ways. Sometimes they are used in the Bookmark standard setting procedure to create an ordered item booklet (OIB) and other times they are used for mapping items to a reporting scale. The PACER software program can estimate the response probabilities for a set of items and return a table with the RP values for each item, a table ordering all items by the RP value, and a collection of visual displays that might be useful to examine the items. The items can be all binary, al polytomous, or a mixture of the two.

The items can be all binary, al polytomous, or a mixture of the two. All binary items are treated as a special case of the 3-parameter logistic model and the polytomous items must be the generalized partial credit model, or a special case of that model (e.g., Master's Partial Credit). The graded response model is coming soon.

In this blog post I will show how to use PACER to estimate the RP values and examine a set of visual displays using data that can be accessed here.
